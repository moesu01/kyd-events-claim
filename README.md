# KYD Events + Claim

Mobile event ticketing and ticket claim flows for KYD Labs — built with Vite, React, TypeScript, and Chakra UI v3.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Flows

- **Event page** — browse event, select tickets, purchase CTA
- **Claim page** — StubHub transfer claim with mock OIDC sign-in dialog

Navigate between flows via the **My Tickets** header dropdown.
